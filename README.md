# STM32 学习记录

STM32 学习仓库！这个仓库主要记录我在学习 STM32 时所遇到的各种实验、代码示例、项目和学习资源。
通过这个仓库，我整理了自己在 STM32 开发过程中获得的知识。

## 目的

这个仓库的目的是：

- 记录和分享我在学习 STM32 时的代码和资源。
- 在此过程中不断优化自己的编程技能和硬件控制能力。

## 学习内容

- 基本外设的驱动程序（GPIO, UART, ADC, PWM 等）
- 高级应用（RTOS, 通信协议等）
- 一些常见问题的解决方案和调试技巧
